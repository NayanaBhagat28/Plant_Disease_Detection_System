## Overview
This is a Final Year AI Project that detects plant diseases from leaf images using a CNN deep learning model trained on the PlantVillage dataset.

## Features
- Upload leaf image
- Predict plant disease (38 classes)
- Show basic treatment recommendations

## Tech Stack
- Python
- TensorFlow / Keras
- Streamlit
- PlantVillage Dataset

## How to Run

### Install dependencies
```bash
pip install -r requirements.txt
```

### Train the model
```bash
python train_model.py
```

### Run the application
```bash
streamlit run app.py
```

## Output
- Disease name displayed
- Recommendation shown
